What this page shows

This page is the individual fryer view. It shows the current status of one fryer, when it was last boiled out, a small history log, and the actions a user or leader would normally take. In this public preview, actions are visible for evaluation, but no data is actually saved.

How the NFC tap would work

In a live version, tapping the NFC tag on this fryer would open this exact page directly. That means the user would not need to search for the fryer manually.
