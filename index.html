What this page shows

This dashboard shows all fryers in one place and gives each fryer a current status based on its last recorded boil-out date and rule settings. The goal is to make it easy for a leader to quickly see what is okay, what is due soon, and what needs attention.

How the NFC tags would work

In a live version, each fryer would have a small NFC tag. A team member would tap the tag with their phone, and that exact fryer page would open instantly. That means no searching, no paper log, and no guessing which fryer is being updated.

How it works

1
Tap fryer
An NFC tag on the fryer opens the correct fryer page immediately.
2
Log or flag
The user logs a completed boil-out or flags that a boil-out is needed.
3
Leadership sees it
Leadership can see the current status of all fryers in one dashboard.
Tap a fryer to view a more detailed preview of how logging, history, and leader tools would work.

How this demo works

This preview is meant to show the basic workflow, not a live saving system.

  • Tap a fryer to open its individual page.
  • In a live version, this would happen from an NFC tag on the fryer.
  • You can explore the actions and leader tools, but this public preview does not save changes.
